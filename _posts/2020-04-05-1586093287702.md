# 利用欧几里得算法求最大公约数

假定输入的第一个参数不小于第二个参数，其递推公式为

\begin{equation}gcd(m, n)=gcd(n, m\ mod\ n)\end{equation}

当n为0时，m即为最大公约数，代码如下

```javascript
function gcd(fst, scd) {
    if (scd == 0) return fst;
    return gcd(scd, fst % scd);
}
```
