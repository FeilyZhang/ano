## 检查序列元素是否唯一

最简单的办法就是遍历，时间复杂度为$n^2$，如下

```javascript
function unique(arr) {
    for (var i = 0; i < arr.length - 1; i++) {
        for (var j = i + 1; j < arr.length; j++) {
            if (arr[i] == arr[j]) {
                return false;
            }
        }
    }
    return true;
}
```
